# EirMed

EirMed is an open-source personal medical record system to centralize exams, consultations, diagnoses, and medication management in one place.

## Project Structure

```
eir-med/
├── backend/          # .NET 9 API (Clean Architecture)
├── frontend/         # Next.js 16 + React 19 + Tailwind CSS
└── docs/             # Project documentation
```
